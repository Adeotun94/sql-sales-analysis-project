# SQL Sales Analysis Project

This project demonstrates SQL data analysis using MySQL and visualizes the results in a spreadsheet. The goal is to extract key business insights from sales data, including revenue trends, top customers, and product performance.

## Key Insights
- Total Revenue generated from all sales
- Revenue per Customer
- Monthly Revenue Trends
- Top Selling Product

## Tools & Skills Used
- MySQL for querying and aggregating data
- SQL skills: SELECT, SUM, GROUP BY, JOINs
- Google Sheets / Excel for data visualization and charts
- Data cleaning and organization

## Project Overview
1. Wrote SQL queries to analyze sales data and calculate metrics such as total revenue, top customers, and monthly revenue trends.  
2. Exported query results into Google Sheets and created charts to visualize the data.  
3. Compiled a summary sheet highlighting key insights for easy interpretation by stakeholders.

## Files Included
- `sales_queries.sql` → contains all SQL queries used in the project  
- `sales_analysis_results.xlsx` → spreadsheet with query results, charts, and summary sheet  

## Project Goal
This project showcases the ability to transform raw data into actionable insights using SQL and spreadsheet tools, which is a key skill for entry-level data analyst roles.
